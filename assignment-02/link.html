Must-Know Web Dev Terms:

HTML
HTML is basic language used to create web pages. it works with tags and elements to organize content like text, images, links, and headings. Every website starts with HTML because it gives the page its structure. Browsers reads HTML to understand what to display and where to place it. Without HTML, a web pages would just be plain text with no format. It is simple to learn and is the first step for anyone starting Web-Development.
CSS
CSS ( Cascadind Style Sheets) It's a language used to style and design web pages.It's used to make it look beautiful. With CSS, we can change the color, fonts, size, spacing, and layout of diffrent elemnets on a web page. It helps in making websites more attractive and user-friendly. CSS also allows us to create responsive design that look good on both mobile phones and computers. Without CSS, website would look plain and boring. That's why CSS is very important im modern web development
JavaScript
JavaScript is a programming language used to make website interactive. We can create things likes image sliders, buttons that do somethings when click, Pop-ups messages, and live form validation. It runs directly in the browser, so user don't need to download anything. Its used almost every modern website and is an important part of Web-Development. Without JavaScript, would be static and less engaging
